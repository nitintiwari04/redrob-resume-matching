# Resume Matching Engine

This project was developed by Nitin for the Redrob AI Campus Hackathon.

## Project Overview
The Resume Matching Engine analyzes candidate resumes and matches them with job descriptions using:

- Skill normalization
- Alias mapping
- TF-IDF vector generation
- JD binary vector construction
- Cosine similarity scoring
- Candidate ranking system

The system processes noisy resume data, standardizes technical skills, and identifies the best candidates for each job role.

## Features
- Resume skill normalization
- Duplicate skill removal
- Shared vocabulary creation
- Manual TF-IDF computation
- Binary vector generation for job descriptions
- Cosine similarity calculation
- Top candidate ranking with tie-breaking

## Technologies Used
- Python
- Standard Python libraries only (`math`)
